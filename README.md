# SBA318 Express Server Application - Quote Collection App

A quote management application built with Node.js and Express. Browse, filter, and add inspirational quotes from famous authors across different categories.

## How to start app:
enter npm install in terminal
enter npm start in terminal
view app in browser: http://localhost:3000

## Project Overview

## 3 Different data categories:
- **Quotes** - Inspirational quotes with text, author, category, and metadata
- **Authors** - Famous people who said/wrote the quotes
- **Categories** - Topics like Motivation, Wisdom, Success, Life, Love, and Humor
